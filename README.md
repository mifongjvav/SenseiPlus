# SenseiPlus

为sensei们提供的额外的[CET](https://github.com/Wangs-official/CodemaoEDUTools)功能

## 如何使用？

请在开始使用前运行以下命令安装依赖，文件内不包含CET的依赖，您需要手动安装CET的依赖

```bash
pip install -r requirements.txt
```
